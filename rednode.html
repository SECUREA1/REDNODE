RedNode Excavation Control Center

A single-page excavation-themed chat and streaming hub with seamless history and optional WebSocket connectivity.

Core Features

Excavation-Themed UI

A cohesive design system with rich gold accents and heavy machinery motifs to immerse operators in the RedNode experience.

Persistent Chat History

Retain important crew discussions with automatic local storage that keeps conversations available between sessions.

Multi-Mode Connectivity

Toggle between local-first usage and WebSocket-enabled collaboration to match the needs of on-site and remote teams.

Live Status Indicators

Monitor live operators, broadcast sessions, and excavator commands at a glance with responsive dashboard chips.

Live Streaming Control

OBS-Ready Codes

Generate stream keys for quick integration with OBS Studio and other RTMP-compatible broadcasting tools.

One-Click Broadcast Mode

Expand any stream to a focused, fullscreen layout for supervisors while keeping the mission feed available to the crew.

Collaborative Monitoring

Allow approved operators to request joining, share thumbnails, and engage in stream-specific discussions.
